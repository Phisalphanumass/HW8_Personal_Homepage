💡 The Father of C
Dennis Ritchie

Dennis Ritchie: บิดาผู้สร้างภาษา C และรากฐานระบบปฏิบัติการ Unix

หากไม่มีภาษา C และ Unix โลกของซอฟต์แวร์และฮาร์ดแวร์ในปัจจุบันก็คงไม่เกิดขึ้น ภาษา C คือตัวเชื่อมสำคัญที่ทำให้โปรแกรมเมอร์คุยกับฮาร์ดแวร์ได้อย่างมีประสิทธิภาพและเข้าถึงหน่วยความจำได้โดยตรง เป็นรากฐานของภาษาโปรแกรมยุคใหม่เกือบทั้งหมด

"C is quirky, flawed, and an enormous success."
— Dennis Ritchie

🐧 Open Source Legend
Linus Torvalds

Linus Torvalds: จอมขบถผู้สร้าง Linux และระบบควบคุม Git

ชายผู้เขียนแกนกลางของระบบปฏิบัติการ (Kernel) ขึ้นมาในหอพักนักศึกษา และแจกจ่ายมันให้คนทั่วโลกใช้ฟรี จน Linux กลายเป็นระบบขับเคลื่อนซูเปอร์คอมพิวเตอร์และเซิร์ฟเวอร์ส่วนใหญ่บนโลกนี้ รวมถึง Git ที่เหล่านักพัฒนาซอฟต์แวร์ใช้จัดการเวอร์ชันโค้ดในปัจจุบัน

"Talk is cheap. Show me the code."
— Linus Torvalds

🐍 Readable Code
Guido van Rossum

Guido van Rossum: ผู้สร้าง Python ภาษาแห่งความเรียบง่ายและทรงพลัง

ชายผู้สร้างภาษาโปรแกรมมิ่งที่เน้นความเรียบง่าย สวยงาม และอ่านง่ายเหมือนภาษาอังกฤษทั่วไป Python ถูกออกแบบมาเพื่อลดความยุ่งยากของไวยากรณ์ ทำให้โปรแกรมเมอร์หันไปโฟกัสที่ตัวตรรกะของปัญหาจริงๆ จนกลายเป็นภาษายอดนิยมในวงการ Data และ AI

"If you can't read your code, you can't maintain it."
— Guido van Rossum
